# employee-management-system-django
A web-based Employee Management System built with Django that allows organizations to efficiently manage their employee records. The system includes secure user authentication with login and logout functionality, ensuring only authorized users can access the platform.

🚀 Features

🔐 User Authentication (Login & Logout)
👨‍💼 Add, View, Update and Delete Employee Records
🗄️ Lightweight SQLite Database
📱 Clean and Responsive UI with HTML & CSS
⚙️ Built with Django Function-Based Views (FBV)


🛠️ Tech Stack
TechnologyUsagePythonBackend LogicDjangoWeb FrameworkSQLiteDatabaseHTMLFrontend StructureCSSStyling & Layout

⚙️ Installation & Setup
bash# Clone the repository
git clone https://github.com/yourusername/employee-management-system-django.git

# Navigate to project directory
cd employee-management-system-django

# Install dependencies
pip install -r requirements.txt

# Run migrations
python manage.py migrate

# Start the server
python manage.py runserver

📌 Usage

Open your browser and go to http://127.0.0.1:8000
Login with your credentials
Manage employee records — Add, Edit, View or Delete
